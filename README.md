**Official site & download page:** https://vikasdisale.com/makdi

Makdi v1.0.0 — first public release 🕷

- Full crawler: 16 analysis tabs with Screaming Frog-style sub-filters
- Scored dashboard (0–100) with prioritised fix suggestions
- Internal link suggestions — unlinked mentions with ready-made anchor text
- Crawl comparison (Changes tab), custom source search, image SEO
- URL detail panel with inlinks, outlinks and anchor text
- Excel action plan + printable PDF report, CSV export on every view
- Robots.txt aware · sitemap mode · JSON-LD validation · fully Unicode-aware

**Windows:** download `MakdiSetup-1.0.0.exe` below and run it.
**Mac/Linux/developers:** clone the repo and run `python makdi.py` (needs `pip install requests beautifulsoup4 lxml openpyxl`).
